# Problem 1

### Read in NY NOAA data

``` r
data("ny_noaa") 
ny_noaa = ny_noaa |>
  janitor::clean_names() |> 
  separate(date, c('year', 'month', 'day'), sep ='-') |>
  mutate(prcp = as.numeric(prcp),
         tmax = as.numeric(tmax),
         tmin = as.numeric(tmin),
         prcp = prcp / 10,
         tmax = tmax / 10,
         tmin = tmin / 10)
```

Count the snow fall

snowfall_counts = ny_noaa |>
